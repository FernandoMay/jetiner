# Jetiner

Travel/journey management platform built with Laravel and Vue.

## Stack

- **Laravel 8** — PHP backend with Eloquent ORM
- **Vue 2** — Frontend via Inertia.js
- **Laravel Mix** — Asset compilation
- **Tailwind CSS 2** — Utility-first styling

## Quick Start

```bash
composer install
npm install
cp .env.example .env
php artisan key:generate
php artisan migrate
npm run dev
php artisan serve
```
